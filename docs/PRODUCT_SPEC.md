# Product Spec — ScanAdmin

## Produit

ScanAdmin est un scan IA administratif pour PME.

## Problème

Les dirigeants de PME perdent du temps sur des tâches administratives répétitives sans savoir combien cela leur coûte réellement.

## Solution

Un formulaire rapide qui estime :

- le volume de tâches administratives ;
- le temps perdu par mois ;
- le coût mensuel ;
- le coût annuel ;
- le temps récupérable avec agents IA ;
- les agents IA recommandés.

## Utilisateur cible

Dirigeant, responsable administratif, office manager, artisan, responsable d'agence ou consultant indépendant.

## Résultat attendu

À la fin du scan, l'utilisateur doit comprendre :

1. où il perd le plus de temps ;
2. combien cette perte coûte ;
3. quelles tâches automatiser en priorité ;
4. quels agents IA supervisés peuvent être utiles ;
5. comment demander une analyse personnalisée.

## Ton commercial

Simple, concret, chiffré, sans jargon.

## Phrase clé

Combien votre administratif vous coûte-t-il chaque mois ?
