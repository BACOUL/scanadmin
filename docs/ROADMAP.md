# Roadmap — ScanAdmin

## V0 — Cadrage

- Valider le positionnement.
- Créer le repo.
- Documenter l'offre.
- Définir les calculs.

## V1 — MVP scan gratuit

- Landing page.
- Formulaire de scan.
- Calcul des pertes administratives.
- Page résultat.
- Agents recommandés.
- CTA vers analyse personnalisée.

## V2 — Capture commerciale

- Formulaire de contact.
- Envoi du résultat par email.
- Exemple de rapport PDF.
- Page offre audit express.

## V3 — Audit express payant

- Paiement simple.
- Génération d'un rapport personnalisé.
- Prise de rendez-vous.

## V4 — Agents IA supervisés

- Modèles d'agents réutilisables.
- Agent mails.
- Agent devis.
- Agent relance.
- Agent documents.

## V5 — Produit SaaS

- Compte client.
- Historique des scans.
- Comparaison avant/après.
- Catalogue d'agents activables.
