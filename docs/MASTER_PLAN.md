# Master Plan — ScanAdmin

## Vision

Créer un outil simple qui chiffre les pertes de temps administratives des PME et recommande les agents IA supervisés les plus rentables.

## Promesse

Découvrez combien vos tâches administratives répétitives vous coûtent chaque mois, et quels agents IA peuvent vous faire gagner du temps et de l'argent.

## Cible prioritaire

PME avec beaucoup d'administratif : BTP, services, nettoyage, transport, formation, e-commerce et cabinets administratifs.

## Angle de différenciation

ScanAdmin ne vend pas d'abord une solution IA. Le produit mesure d'abord la perte administrative : temps perdu, coût mensuel, coût annuel, puis agents recommandés.

## Tunnel commercial

1. Scan gratuit.
2. Résultat chiffré immédiat.
3. Demande d'analyse personnalisée.
4. Audit express payant.
5. Installation d'un agent IA supervisé.
6. Maintenance mensuelle.

## Priorités V1

- Landing page.
- Formulaire de scan.
- Calculateur local.
- Page résultat.
- Catalogue initial des agents IA.
- Documentation produit.
